# 객체 리터럴

## 객체

- JS는 객체 기반의 프로그래밍 언어
  - JS를 구성하는 거의 '모든 것'이 객체
  - 원시값을 제외한 나머지 값(함수, 배열, 정규 표현식 등)은 모두 객체이다.
- 원시값은 변경 불가능 한 값이지만, 객체 타입의 값은 변경 가능한 값이다.
- JS의 함수는 '일급 객체'이므로 값으로 취급 가능하다.
  - 프로퍼티 값이 함수일 경우, 이를 **`메서드`** 라고 부른다.

### 객체의 구성

1. 프로퍼티
   - 객체의 상태를 나타내는 값(data)
2. 메서드
   - 프로퍼티를 참조하고 조작할 수 있는 동작
