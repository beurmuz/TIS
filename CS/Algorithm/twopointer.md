# 부분합

- N개의 수 중, i번째 수부터 j번째 수까지의 합?
  - Sum(N, M): N번 인덱스 ~ M번 인덱스까지의 합
  - Sum(1, M): 1번 인덱스 ~ M번 인덱스까지의 합
  - Sum(N, M): arr[N] + arr[N+1] + ... + arr[M]
    - Sum(1, M) - Sum(1, N-1)

# Two Pointer (투포인터)

- 1차원 배열에서 두 개의 포인터를 조작해 답을 얻는 방법
- 모든 배열을 순회할 필요 없이, 포인터 두개로 O(N)만에 탐색한다.
